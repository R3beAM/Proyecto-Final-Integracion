# Documentación

Esta carpeta reúne documentos de apoyo para el **Proyecto Final de Integración**, incluyendo guías, entregables exportados, scripts derivados de notebooks y notas técnicas que complementan el código fuente, los datos y los reportes del repositorio.

## Contenido

- `proyecto_final_integracion_(1)_(1).py`: versión en Python/exportación del desarrollo metodológico del proyecto.
- `test`: archivo auxiliar reservado para pruebas o validaciones temporales.

## Uso recomendado

1. Mantener aquí la documentación técnica y metodológica que explique decisiones del proyecto.
2. Guardar guías de configuración, operación o reproducción que no pertenezcan a notebooks, datos, código fuente o reportes finales.
3. Usar nombres descriptivos para nuevos documentos y evitar archivos temporales sin contexto.
4. Actualizar este README cuando se agreguen, eliminen o reorganicen documentos relevantes.

## Relación con otras carpetas

- `Notebooks/`: notebooks exploratorios y entregables ejecutables.
- `src/`: código fuente reutilizable.
- `data/`: estructura de datos administrada con DVC.
- `reports/`: salidas finales, figuras, PDFs y reportes de monitoreo.
- `mlflow_tracking/`: documentación y scripts relacionados con experimentos de MLflow.
